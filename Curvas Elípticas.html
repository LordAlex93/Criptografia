Curvas Elípticas

1. Dar un ejemplo de una curva elíptica sobre \(\mathbb{R}\) con exactamente dos puntos de orden \(2\) y otra con exactamente \(4\) puntos de orden \(2\).

Una curva elíptica sobre \(\mathbb{R}\) con exactamente dos puntos de orden \(2\) es: \[y^2=x^3-125,\] hay dos puntos para \(y=0\) que son \((5,0)\) y \((-5,0)\) que además son opuestos entre sí.

Una curva elíptica sobre \(\mathbb{R}\) con exactamente cuatro puntos de orden \(2\) es: \[y^2=x^3-n^2x,\] para cualquier \(n\) nos quedan los puntos \((0,0)\), \((n,0)\), \((-n,0)\) y el infinito.

2. Sea \(P\neq \mathcal{O}\) un punto de una curva elíptica sobre \(\mathbb{R}\). Dar una condición geométrica que caracterice los puntos con orden \(2\), \(3\) y \(4\).

Si tenemos dos soluciones \(P=(x_1,y_1)\in \mathbb{R}^2\) y \(Q=(x_2,y_2)\in\mathbb{R}^2\) de nuestra ecuación \[y^2 = x^3 + Ax^2 + Bx + C,\] entonces calculamos \(P + Q\) de la manera siguiente:

  • Si \(P\neq Q\) son puntos distintos, entonces trazamos la recta que pasa por P y Q. Dado que nuestra curva está dada por una ecuación de grado 3, esta recta interseca la curva elíptica en un tercer punto. Luego, definimos la suma \(P + Q\) como el punto obtenido al considerar la simetría ortogonal de este último punto respecto al eje \(X\).

  • Si \(P = Q\) son el mismo punto, entonces consideramos la recta tangente a la curva elíptica pasando por el punto \(P\). Tal como antes, la recta tangente interseca la curva elíptica en un tercer punto. Luego, definimos la suma \(P+P:= 2P\) como el punto obtenido al considerar la simetría ortogonal de este último punto respecto al eje \(X\).} \end{itemize} Y así sucesivamente podemos obtener puntos de orden \(3\) y \(4\).

    gamma = k*P # Clave Pública de Bob

    x1 = (mp[0],mp[1])
    x2 = (mp[2],mp[3])

    y1 = (x1[0]*c1)%751,(x1[1]*c2)%751

6. Implemente el método de curvas elípticas de Lenstra y factorice los siguientes números:

  • \(N = 199843247\)
  • \(N = 8746327411126827157\)
  • \(N = 10817310890509097139\)
  • \(N = 104681783926243375222281565321123456789\)
Código 
def lenstra_ecm(N, B):
    """
    Implementación del método de curvas elípticas de Lenstra en SageMath.
    N: Número a factorizar.
    B: Límite de primos pequeños para el escalado del punto.
    """
    
    while True:
        # Seleccionar un punto y una curva aleatoria
        a = randint(1, N-1)
        x0 = randint(1, N-1)
        y0 = randint(1, N-1)
        b = (y0^2 - x0^3 - a*x0) % N
        
        try:
            E = EllipticCurve(GF(N), [a, b])  # Crear la curva elíptica sobre Z/NZ
            P = E(x0, y0)  # Punto sobre la curva
        except:
            d = gcd(4*a^3 + 27*b^2, N)
            if 1 < d < N:
                return d  # Factor encontrado
            continue
        
        # Aplicar el escalado por primos pequeños
        for p in primes(B):
            e = int(log(B) / log(p))  # Exponente máximo de p tal que p^e <= B
            P = p^e * P  # Multiplicación en la curva
            
            # Verificar si encontramos un factor
            if P[2] == 0:  # Si la coordenada Z es 0, significa que hubo un problema con N
                d = gcd(P[0], N)
                if 1 < d < N:
                    return d  # Factor encontrado
        
        # Si fallamos, repetimos con otra curva
